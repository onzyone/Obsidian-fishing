## Summary

- **Total: ~30 pins** across 7 creeks (some overlap with canonical Access Points expected — dedupe needed).
- **Richest signal:** Twelve Mile (St. Catharines) and Kettle Creek (Port Stanley) — multiple confirmed bridges/piers/parks. Kettle's Port Stanley harbour densest.
- **Strongest single pin:** Misner Dam, Port Dover (Lynn River) — confirmed by an *anti-fishing petition* about illegal fishing within 75ft of dam (proof of heavy pressure).
- **Weakest signal:** Big Creek (Lake Erie / Long Point) — most named features are sanctuary boundaries (Quance/Teeterville dams) or marsh access. No user-coined GMaps spot.
- **Did NOT find equivalents to "313 fishing spot" style user-named GMaps places** anywhere in Southwest. Plain WebSearch doesn't surface user-named map points.

## Caveats

- **Lake Erie clay basin** = chocolate-milk fast after rain; many pins unfishable 24–72h post-storm.
- **Migratory-only**: Lynn (Misner), Big Otter (below Black's), Catfish (lower below Sparta), Kettle (lower St. Thomas → Port Stanley) — runs late Oct → April. Outside that, mostly warmwater.
- **Twelve Mile is special**: only true year-round cold-water creek — Short Hills upper holds resident brookies, conservation-sensitive.
- **Sanctuaries**: Big Creek (Teeterville → Quance) and Big Otter (Black's → Oxford St) — verify FMZ 16 regs.
- **Misner Dam 75ft rule**: federally illegal to fish within 22.9m / 75ft of dam base.
- **Private property heavy**: Catfish mid-section + most upstream rural reaches of Big Otter and Big Creek run through ag land.
- **Coordinate confidence**: lat/lons derived from named landmarks via map lookup, not user-pinned places. Treat as ±50–200m. Verify before pinning.
