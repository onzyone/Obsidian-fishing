# Fishing Rules

- **Fishing locations** — follow [[52 Fishing Locations/fishing-location-standards|Fishing Location Standards]] whenever adding or editing river/creek notes. After adding a river from `Add these rivers.md`, remove that entry from the file.

- **River queue** — `52 Fishing Locations/Add these rivers.md` is the queue for rivers to be researched and added. When the user asks to add rivers, check this file first. After each river is fully added (notes file + both index tables), remove it from the queue file.
