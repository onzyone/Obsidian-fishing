# Fishing Rules

- **Fishing locations** — follow [[fishing location skill/fishing-location-standards|Fishing Location Standards]] whenever adding or editing river/creek notes. After adding a river from `Add these rivers.md`, remove that entry from the file.

- **River queue** — `fishing location skill/Add these rivers.md` is the queue for rivers to be researched and added. When the user asks to add rivers, check this file first. After each river is fully added (notes file + both index tables), remove it from the queue file.
