# Fishing Location Standards

Whenever adding a new river or creek to `Ontario Trout & Steelhead Locations.md`:

1. Create a dedicated notes file at `Rivers/<Direction>/<River Name>.md` using `Rivers/West/Credit River.md` as the template
2. Directions are: `North`, `West`, `East`, `Southwest` (Lake Erie / Niagara tribs)
3. Use Obsidian wiki link format in the main index table: `[[Rivers/<Direction>/<River Name>|River Name]]`
4. River notes must include: species, drive time, regulations by section (FMZ + open season + limits), access points with parking details and walk distances, public vs. private land notes, run timing, and wading/conditions
5. Always update both the Quick Reference table AND the Best for Catch & Keep table in the main index
